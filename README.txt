# takata-readable-code
◆実行手順◆

１．ソースファイル一式をGithubから取得。
　　フォルダ"src"をPullする。
　　
２．Eclipseでプロジェクトを新規作成する。

３．2で作ったプロジェクトのsrcフォルダへ、1で取得したソースをコピー。具体的な手順は(1)-(3)の順番。
　　(1) 1で取得したフォルダ"src"をエクスプローラで表示。
　　(2) エクスプローラから、Eclipseの「パッケージ・エクスプローラ」上の"src"めがけてソースをドラッグ&ドロップ。
　　(3) 「ファイル操作」ダイアログで[OK]をクリック。　　
　　
４．２で作ったプロジェクトを実行し、動作確認。
